# 🚀 Career Compass

Career Compass is a career guidance web application designed to help students stay consistent in their placement preparation by generating structured daily tasks based on their selected domain and preparation duration.

---

## 📌 Problem Statement

Many students preparing for placements struggle with:

- Not knowing what to study daily
- Lack of structured preparation
- Difficulty maintaining consistency
- No clear progress tracking system

This often results in confusion, low productivity, and inconsistent learning.

---

## 💡 Solution

Career Compass provides a personalized preparation system where users select their domain and preparation duration. The platform generates daily learning tasks, tracks completion, and motivates users through streaks and leaderboards.

---

## ✨ Features

- ✅ User Authentication (Signup/Login)
- ✅ Domain-based learning roadmap
- ✅ Daily task generation
- ✅ Task completion tracking
- ✅ Preparation streak system
- ✅ Daily & All-time Leaderboard
- ✅ Job Application Tracker
- ✅ Community dashboard
- ✅ Announcements and task management
